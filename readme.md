# Laravel Quickstart - Basic

 # Installation :
 ``git clone https://github.com/laravel/quickstart-basic quickstart``
 
 ``cd quickstart``
 
 ``composer install``
 
 ``php artisan key:generate``
 
 ``php artisan migrate``
 
 ``php artisan serve``

 [Complete tutorial](http://laravel.com/docs/quickstart)
